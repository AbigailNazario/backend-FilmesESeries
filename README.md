# App de Mídias (Backend API)

Este é o backend da aplicação **App de Mídias**, responsável pelo gerenciamento de filmes e séries através de uma API REST.


## Funcionalidades

* Listar mídias
* Buscar mídia por ID ou nome
* Criar nova mídia
* Atualizar mídia
* Remover mídia


## Tecnologias utilizadas

* Node.js
* Express
* MongoDB
* Mongoose


## Como rodar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/backend.git
```

2. Acesse a pasta:

```bash
cd backend
```

3. Instale as dependências:

```bash
npm install
```

4. Configure o arquivo `.env`:

```env
MONGO_URI=sua_string_do_mongodb
PORT=3000


## Estrutura do projeto

backend/

```
